# McScraggins

An [Alexa](https://alexa.amazon.com/spa/index.html) skill requested by my daughter Zeyana.  
```
alexa, do the scraggins.

McScraggins, McScraggins, McScraggins.
```
Built using [jovo](https://github.com/jovotech/jovo-framework)

![img](scraggins.jpg)

### build

```bash
npm run bundle
jovo build
```

### run local

```bash
jovo run
```

### deploy

```bash
jovo deploy
```
